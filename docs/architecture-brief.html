Version 1.0
Target host my.autocare.org/wrapped
Repo Autocarewrapped (React + Vite)

Purpose

Auto Care Wrapped is a company-scoped, login-protected “year in review” experience. Each authenticated member organization sees metrics and narrative copy derived from Snowflake. The browser never connects to Snowflake or stores Snowflake credentials.

Request flow: user visits Wrapped → SSO cookie validated → API maps Impexium org ID → Snowflake company_id → returns WrappedReport JSON

Security rule: Company ID is resolved server-side from SSO claims only. The client never sends or selects company_id. Snowflake credentials live in server secrets only.

Frontend (this repository)

Key modules

  • AuthGate — loading, 401, 404, error states
  • WrappedReportContext — fetch once, share via React context
  • DrivingView — landing, journey, hood, diagnostics scenes
  • contentVariants.ts — conditional copy (e.g. zero events)
  • buildJourneySections.ts — report → UI sections
UI + variants Mock data Dev API plugin

Backend (Auto Care stack)

API responsibilities

  • Validate Impexium SSO session (cookie or BFF)
  • Map Impexium org ID → Snowflake company_id
  • Query pre-aggregated secure view via Snowpark / JDBC
  • Map row → WrappedReport JSON; return 404 if no row
  • Optional: 5–15 min cache per company; audit log access
Snowflake view SSO client Prod API deploy

Data contract — WrappedReport

company id, name; optional member.displayName Intro welcome, report scope
