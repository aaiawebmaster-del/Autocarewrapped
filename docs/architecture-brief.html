Purpose

Auto Care Wrapped is a company-scoped “year in review” experience for member organizations. Each company sees engagement metrics, narrative copy, and CTAs to Auto Care products and events. Data is scoped by Impexium RECORDNUMBER (company ID). The browser never connects to Snowflake directly.

Deployment phases

Phase 1 — Live today (Netlify embed)

  • Netlify hosts built SPA + static JSON + embed.js
  • Impexium pages at /engagement/{recordNumber} load embed snippet
  • Embed reads path → iframe → app loads /data/reports/{recordNumber}.json
  • Excel workbook → npm run import:reports → JSON at build time
UI complete Excel pipeline Netlify deploy

Phase 2 — Target (my.autocare.org integrated)

  • SPA served from my.autocare.org/wrapped
  • Impexium SSO cookie validated server-side
  • BFF API queries Snowflake secure view per session org
  • Factbook & TrendLens® via separate product APIs
Snowflake view SSO client Prod API deploy

Frontend (this repository)

Key modules

  • AuthGate — loading, 401, 404, error states
  • WrappedReportContext — fetch report once via context
  • DrivingView — orchestrates all journey scenes
  • buildJourneySections.ts — report → UI sections
  • contentVariants.ts — conditional copy & CTAs
  • externalCtaLinks.ts — centralized external URLs
  • import-excel-reports.mjs — Excel → WrappedReport JSON

Backend (Auto Care stack)

API responsibilities (Phase 2)

  • Validate Impexium SSO session (cookie or BFF)
  • Map Impexium org ID → Snowflake company_id
  • Query pre-aggregated secure view via Snowpark / JDBC
  • Map row → WrappedReport JSON; return 404 if no row
  • Ref implementation: server/reference-api.mjs

Data contract — WrappedReport

Domain Fields (source → SPA) UI surfaces
company id, name; optional member.displayName Intro welcome, report scope
journey Tenure, active contacts, community, committee counts Your Journey counters, Full Diagnostics row
events In-person attended/total, attendancePct, webinar hours GPS nav, attendance route track (variant copy if 0 events)
products TrendLens®, Demand Index, Factbook, Academy usage Kick the Tires readout hubs + external CTAs
standards Subscribed %, missing product list Under the Hood standards popup
